# Enable Chrome Gemini

A Codex skill to set up or repair Gemini in Chrome across Windows, macOS, and Linux.

## What this skill does

This skill enables Google's Gemini AI assistant in Chrome browser, allowing you to use Gemini for:
- AI-powered search assistance
- Content summarization
- Writing help
- Code explanations

## Supported Platforms

- **Windows** - Full support
- **macOS** - Full support
- **Linux** - Full support

## Usage

In Claude Code, invoke the skill:

```
/enable-chrome-gemini
```

The skill will:
1. Detect your operating system
2. Check Chrome installation and version
3. Enable Gemini features in Chrome settings
4. Verify Gemini is working properly

## Prerequisites

- Chrome browser installed (version 120+ recommended)
- Google account signed in to Chrome
- Internet connection

## Troubleshooting

If Gemini doesn't appear after running the skill:
1. Ensure you're signed into Chrome with a Google account
2. Update Chrome to the latest version
3. Check if Gemini is available in your region
4. Try restarting Chrome
